# 📚 Document Intelligence Platform

## Features
- Book CRUD system (Django REST API)
- Smart genre classification
- AI-style book summaries
- Question answering system
- Recommendation engine
- Frontend dashboard (HTML + Tailwind)

## Tech Stack
- Backend: Django REST Framework
- Frontend: HTML, JavaScript, Tailwind CSS
- Database: SQLite

### Frontend

## Sample Features
- Add books
- View books
- Ask questions about books
- Get recommendations

## API Endpoints
- /api/books/
- /api/books/<id>/
- /api/ask/
- /api/recommend/<id>/
